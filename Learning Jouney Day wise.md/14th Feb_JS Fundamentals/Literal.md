===============================
LITERALS IN JAVASCRIPT
===============================

1) DEFINITION
--------------

A literal is a fixed value written directly in the code.

Simple Meaning:
Literal = Actual value you type in the program.


--------------------------------------------------

2) EXAMPLE
-----------

let age = 25;

let      → Keyword
age      → Identifier
25       → Number Literal

Here, 25 is a literal because it is written directly in the code.


--------------------------------------------------

3) TYPES OF LITERALS
---------------------

A) Number Literal
------------------

Definition:
Numeric values written directly in code.

Examples:
let price = 100;
let pi = 3.14;
let temp = -5;

100, 3.14, -5 → Number literals


--------------------------------------------------

B) String Literal
------------------

Definition:
Text enclosed in quotes.

Examples:
let name = "Yajna";
let city = 'Delhi';
let msg = `Hello`;

"Yajna", 'Delhi', `Hello` → String literals


--------------------------------------------------

C) Boolean Literal
-------------------

Definition:
Logical true or false value.

Examples:
let isActive = true;
let isAdmin = false;

true, false → Boolean literals


--------------------------------------------------

D) Null Literal
----------------

Definition:
Intentional empty value.

Example:
let data = null;

null → Null literal


--------------------------------------------------

E) Undefined Literal
---------------------

Definition:
Represents an undefined value.

Example:
let value = undefined;

undefined → Undefined literal


--------------------------------------------------

F) Object Literal
------------------

Definition:
Collection of key-value pairs inside curly braces.

Example:
let person = {
    name: "John",
    age: 25
};

Entire { name: "John", age: 25 } → Object literal


--------------------------------------------------

G) Array Literal
-----------------

Definition:
Collection of values inside square brackets.

Example:
let numbers = [1, 2, 3];

[1, 2, 3] → Array literal


--------------------------------------------------

4) IMPORTANT UNDERSTANDING
---------------------------

Literal = Direct value
Identifier = Name
Variable = Memory container

Example:

let a = 10;   // 10 is literal
let b = a;    // a is identifier (not literal)


--------------------------------------------------

5) QUICK SUMMARY TABLE
-----------------------

--------------------------------
